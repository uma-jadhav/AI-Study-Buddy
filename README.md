# AI Study Buddy

##  About the Project

Built under **Theme 1: Sustainability & Social Impact (Education & EdTech)** for the Idea2Impact Hackathon 2026. This tool aims to make quality education and self-study support accessible to every student, helping them learn efficiently without needing expensive personal tutors.

AI Study Buddy is an AI-powered learning assistant that helps students learn more effectively. It provides instant answers, summarizes notes, generates quizzes, and gives personalized study tips using the Google Gemini AI model.

-  Ask AI

##  Installation

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Add your Gemini API key in `config.env`

```
GEMINI_API_KEY=YOUR_API_KEY
```

4. Run the project

```bash
python app.py
```
